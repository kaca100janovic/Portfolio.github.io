My Projects

Insurance Application

Technologies: Playwright, SQL, Azure DevOps I worked on a web application for an insurance company, which gave me hands-on experience in the insurance domain. My responsibilities included writing and executing manual test cases, identifying and documenting bugs, and creating automated end-to-end tests using Playwright. I also wrote SQL queries to validate that data was correctly populated across all relevant fields. Testing required a high level of precision and attention to detail, as the application serves multiple international markets, each with its own set of business rules. I used Azure DevOps for test management and tracking. In addition to testing activities, I was responsible for preparing and delivering presentations to both clients and internal business teams, showcasing new system functionalities and improvements. These sessions were held in English and involved translating technical updates into clear, user-focused demonstrations.

Corporate Travel Management Portal

Technologies: Jira, Swagger, Manual Testing The goal of this project was to build a new portal that provides companies with insights into their employees' travel data, allows streamlined travel-related payments, and includes CO2 emissions tracking features. I was responsible for writing test plans and manual test cases, and for working with Swagger to understand and validate APIs. I also maintained detailed bug statistics — tracking where bugs occurred and which user segments were affected. The project required close collaboration with business and marketing teams, as we frequently received client feedback and translated it into actionable improvements to enhance the user experience.

ORDER FOOD

Technologies: Manual Testing, UI Validation, Bug Reporting The application is a food ordering and management platform designed for two user groups: employees and food providers. Employees can browse menus and place orders, while providers manage food items, track orders, and review user feedback. My role on this project involved writing and executing manual test cases, ensuring that core features such as order placement, menu updates, and feedback handling functioned as expected. I performed detailed UI validation, reported bugs, and collaborated with the team to continuously improve the user experience for both types of users.
